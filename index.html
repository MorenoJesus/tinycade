What Tinycade Is

Tinycade is a modular launcher and game collection built for an ESP32 handheld setup. It treats the board like a miniature arcade cabinet: fast boot, quick game switching, shared controls, and lots of pick-up-and-play action.

Micro Arcade Vibe

The goal is not to imitate a giant game console. It is to feel like a tiny glowing cabinet that always has one more run left in it. Fast menus, bold visuals, simple controls, and games that feel good in short bursts are the heart of the project.

Two Buttons, Many Games

Tinycade intentionally keeps the hardware simple. Two buttons handle menu navigation, launching games, and all in-game actions. That constraint shapes the design of every game and gives the whole console its identity.

Cabinet Specs

Under the hood, Tinycade is a compact ESP32 launcher with a shared rendering loop, a unified game interface, and enough retro attitude to make a tiny screen feel larger than it is.

Core Hardware

  • ESP32 microcontroller
  • 135×240 TFT display
  • Two physical buttons
  • Backlit handheld-style form factor

Shared System

  • Single launcher menu for all games
  • Shared sprite rendering pipeline
  • Consistent game module interface
  • Global in-game exit with both buttons held

Why It Works

Tinycade feels good because it embraces its limits. Two buttons, one small display, fast game loops, bright arcade identity, and a launcher that makes the whole thing feel like a real little console instead of a pile of unrelated sketches.
